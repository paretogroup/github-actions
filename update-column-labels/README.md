# Update column labels

Updates the column label of an issue

## Inputs

### `token`

**Required** The token.

### `issue_id`

**Required** The issue id.

### `all_column_names`

**Required** All the columns name.

### `column_name`

**Required** The column name.

### `organization_name`

**Required** The organization name.

### `repository_name`

**Required** The repository name.
